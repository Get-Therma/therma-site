# Sound Files for Penguin Intro Animation

This directory contains sound effects for the penguin intro animation.

## Required Files

1. **slide1.mp3** - Penguin sliding sound effect (whoosh/slide noise)
2. **slide2.mp3** - Alternative penguin sliding sound (for variety)
3. **ambient.mp3** - Ambient city/street sounds (subtle background)

## Recommended Sources for Free Sound Effects

### Freesound.org
- Search for "slide", "whoosh", or "ice slide"
- Search for "city ambience" or "street sounds"
- License: Creative Commons (check individual sounds)

### Zapsplat.com
- Professional quality sound effects
- Free with attribution
- Search: "cartoon slide", "whoosh", "winter ambience"

### Mixkit.co
- Royalty-free sound effects
- No attribution required
- Search: "whoosh", "slide", "city"

## File Specifications

- **Format**: MP3
- **Size**: Keep under 100KB each for fast loading
- **Length**: 
  - Slide sounds: 0.5-2 seconds
  - Ambient: 10-30 seconds (will loop)
- **Bitrate**: 128kbps is sufficient

## Notes

- All sounds are **optional** - the animation works without them
- Sounds are **muted by default** with an unmute button
- The animation will gracefully handle missing audio files

## Installation

1. Download your chosen sound files
2. Rename them to match the required filenames above
3. Place them in this directory (`public/sounds/`)
4. Reload the page to hear the sounds (unmute button)
